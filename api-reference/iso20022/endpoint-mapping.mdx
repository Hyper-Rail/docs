#### Field Mapping

| JSON Field | ISO 20022 Path | Description |
| :--- | :--- | :--- |
| **rate** |TradInf/XchgRate | Exchange rate |
| **from_currency** | TradInf/TradgPty/SellCcy | Source currency |
| **to_currency** | TradInf/TradgPty/BuyCcy | Destination currency |
| **ttl_seconds** | FxXchgDtls/VldtyPrd | Validity period (TTL) |
| **available_liquidity** | SplmtryData/Envlp/Amt  | Available liquidity amount |
