# Introduction

<Badge text="Payments Application" />
<Badge text="Fiat & Stablecoins" />
<Badge text="Liquidity Marketplace" />

HyperRail is an **exchange and liquidity orchestration application** that allows businesses to move value between **fiat currencies and stablecoins** across global payment corridors.

It enables **on-ramping, off-ramping, and fiat-to-fiat exchange** using stablecoins as the settlement layerâ€”while handling pricing, liquidity sourcing, and settlement internally.

HyperRail is built for businesses that need **fast, reliable, and transparent currency exchange**, without managing banks, blockchains, or liquidity providers directly.

---

## Who Uses HyperRail
  <Card title="Merchants">
    Businesses that convert currencies, on-ramp or off-ramp funds, and execute trades using available liquidity.
  </Card>

  <Card title="Liquidity Providers">
    Institutions or partners that supply liquidity to currency pairs and earn margins on executed trades.
  </Card>

---

## What You Can Do

- Convert fiat to stablecoins and back
- Exchange between fiat currencies via stablecoins
- View real-time exchange rates before trading
- Manage balances and transaction history
- Access liquidity from multiple providers through one interface

---

## How HyperRail Works (Quick View)

<Steps>
  <Step title="Liquidity Is Supplied">
    Liquidity Providers fund currency pairs and set exchange pricing.
  </Step>

  <Step title="Rates Are Generated">
    HyperRail aggregates liquidity and displays the best available rates.
  </Step>

  <Step title="Trades Are Executed">
    Merchants initiate exchanges and settle instantly using available liquidity.
  </Step>
</Steps>

## Getting Started

### For Merchants
1. Create a HyperRail account at [https://app.hyperail.io](https://app.hyperail.io/signup) 
2. Select merchant as service type
3. Complete business verification  
4. Fund your account 
5. Start exchanging currencies  

### For Liquidity Providers
1. Create a HyperRail account at [https://app.hyperail.io](https://app.hyperail.io/signup) 
2. Select liquidity provider as service type
2. Complete onboarding and approval  
3. Fund supported currency pairs  
4. Set rates and monitor performance
